Data structure

class
structured_rep.Item(dic)

Represents the basic building blocks of the structured representations - objects that have shape, color and spatial location, each belongs to a certain 'box'.

Attributes
  • bottom — Return the bottom coordinate of the item.
  • left — Return the left coordinate of the item.
  • right — Return the right coordinate of the item.
  • top — Return the top coordinate of the item.

  • touching_right(use_margin) — Check if the item is touching the right.
  • touching_top(use_margin) — Check if the item is touching the top.
  • touching_wall(use_margin) — Check if the item is touching a wall.
